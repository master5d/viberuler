## LinkedIn

I shipped a small open-source benchmark this week: viberuler.

One command — `npx viberuler` — scans your AI-assisted coding activity locally (Claude Code / Codex logs + git history) and computes a score around one metric I think matters more than any other in this new workflow: **tokens per dollar**. Volume is easy; efficiency is the skill.

Design constraints I held: zero network calls by default, aggregates-only opt-in leaderboard with the exact payload printed before send, and a fully published methodology. The rank names are memes; the measurement isn't.

If you build with AI daily, I'd genuinely like to see your number. And if your agent of choice isn't supported yet — the collector interface is two methods, PRs welcome.
